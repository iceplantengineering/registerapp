# レジ操作ガイド (Register Operation Guide)

スマートフォンで片手で簡単にレジ操作を確認できるWebアプリです。

## 概要

スーパーマーケットのレジ研修チェックリスト（PDF全11ページ）をデジタル化し、  
スマホで素早く操作手順を検索・閲覧できるアプリとして再構築しました。

## 機能

- 📂 **11カテゴリ・150+項目** のレジ操作手順を収録
- 🔍 **全文検索** — No.・キーワードで瞬時に検索
- 👆 **スワイプ操作** — 詳細画面で左右スワイプで前後移動
- 🌙 **ダーク/ライトモード** — ワンタップで切り替え
- 📱 **スマホ最適化** — 片手操作に最適なUI設計

## ファイル構成

```
registerapp/
├── index.html        # メインアプリ（レジ操作ガイド）
├── app-data.js       # 操作データベース（全150+項目）
├── app-logic.js      # アプリロジック（画面遷移・検索・スワイプ）
├── index2.html       # 旧試作品（研修ナビ・完了確認機能付き）
├── scan-001.pdf      # 元PDF（横向き）
├── scan-002.pdf      # 90度回転済みPDF（正位置）
├── pdf_pages/        # PDFページ画像（高解像度）
├── pages/            # ページ画像
├── pages_hd/         # 高精細ページ画像
└── README.md
```

## 使い方

`index.html` をブラウザで開くだけで使用できます。  
サーバー不要の静的HTMLアプリです。

## 操作バッジの意味

- 🔵 **レジ操作 (R)** — レジ本体での操作
- 🟡 **J-Mups端末 (J)** — J-Mups端末での操作
- 🔴 **お客様/PayPay (P)** — お客様側・PayPay関連の操作
