# レジ操作ガイド (Register Operation Guide)

スマートフォンで片手で簡単にレジ操作を確認できるWebアプリです。

## 概要

スーパーマーケットのレジ研修チェックリストをデジタル化し、  
スマホで素早く操作手順を検索・閲覧できるアプリとして再構築しました。

## 機能

- 📂 **11カテゴリ・178項目** のレジ操作手順を収録
- 🔍 **全文検索** — No.・キーワードで瞬時に検索
- 👆 **スワイプ操作** — 詳細画面で左右スワイプで前後移動
- 🌙 **ダーク/ライトモード** — ワンタップで切り替え
- 📱 **スマホ最適化** — 片手操作に最適なUI設計

## 使い方

`index.html` をブラウザで開くだけで使用できます。  
サーバー不要の静的HTMLアプリです。

## 操作バッジの意味

- 🔵 **レジ操作 (R)** — レジ本体での操作
- 🟡 **J-Mups端末 (J)** — J-Mups端末での操作
- 🔴 **お客様/PayPay (P)** — お客様側・PayPay関連の操作
- 🩵 **タブレット・PC (T)** — タブレット・PCでの操作

## データ更新手順

1. `data.csv` を編集（CSV形式: 大分類, 中分類, No., 研修項目, 操作手順・内容, 備考欄）
2. 変換スクリプトで `app-data.js` を再生成
3. ブラウザで `index.html` を開いて動作確認
